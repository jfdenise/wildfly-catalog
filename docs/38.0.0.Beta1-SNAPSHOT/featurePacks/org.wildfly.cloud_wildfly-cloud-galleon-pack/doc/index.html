Documentation for WildFly Galleon feature-pack for the Cloud

This feature-pack adjusts the WildFly layers to run on the cloud.

Maven Coordinates

        
<dependency>
  <groupId>org.wildfly.cloud</groupId>
  <artifactId>wildfly-cloud-galleon-pack</artifactId>
  <version>8.0.1.Final-SNAPSHOT</version>
</dependency>

Feature Pack Location

        
<feature-pack>
    <location>org.wildfly.cloud:wildfly-cloud-galleon-pack:8.0.1.Final-SNAPSHOT</location>
</feature-pack>
